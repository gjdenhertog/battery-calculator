## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| build → shipped dist/ | Anything a dependency injects at build time crosses into the artifact users load |
| browser → any origin | The shipped page must make zero outbound requests with user data (none in Phase 1; contract established now) |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-01-01 | Tampering | injected third-party `<script>` (XSS) | mitigate | CSP `script-src 'self'`, no `'unsafe-inline'`; verified by dist/index.html grep in Task 3 and Plan 02 csp-plugin test |
| T-01-02 | Information Disclosure | analytics/beacon exfil via connect/img/font | mitigate | CSP `connect-src 'none'` + `img-src 'self'` + `font-src 'self'` (D-06 system fonts); assetsInlineLimit 0 prevents data: URIs |
| T-01-03 | Tampering | inline modulepreload polyfill script | mitigate | `build.modulePreload: { polyfill: false }` (Pitfall 1) — no inline script emitted |
| T-01-04 | Elevation of Privilege | clickjacking via iframe embed | accept | `frame-ancestors 'none'` is advisory-only via `<meta>` (Pitfall 6); GitHub Pages cannot set HTTP headers; no sensitive data in Phase 1 |
