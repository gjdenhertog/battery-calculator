| Boundary | Description |
|----------|-------------|
| uploaded CSV strings ŌåÆ DOM readout | File names + cell-derived strings cross into rendered DOM (XSS surface) |
| CSP relaxation ŌåÆ blob worker | The Plan-01 `worker-src blob:` must not have widened the network surface |
| large file ŌåÆ main thread | A malicious/huge file could freeze the UI (DoS) |
